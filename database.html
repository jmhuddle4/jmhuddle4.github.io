CSC 442 Introduction to Database Management
Oracle Application Express Project
Spring 2015

The goal of the Introduction to Database Management course is to cover the fundamentals of database systems and provide the foundation of knowledge necessary to design, implement, and manage a variety of database systems. Topics covered are file systems and database concepts, database models, relational database model, introduction to SQL, database design and implementation, database integrity, normalization of database tables and implementation techniques using commercial DBMS. The purpose of the project was to get familiar with Oracle Application Express to design database tables, and make a user interface where a user can make queries.
