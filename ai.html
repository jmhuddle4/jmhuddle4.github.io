CSC 542 Artificial Intelligence
Artificially Intelligent Tic Tac Toe C++
Spring 2015

Tic-tac-toe can be played on a general nrRows x nrCols board where a player wins by having nrToWin of its elements consecutively in a row, column, diagonal, or anti-diagonal. The traditional tic-tac-toe corresponds to the case where nrRows = nrCols = nrToWin = 3. This project, implements an evaluation function for the general tic-tac-toe game.

The interface is: nrRows nrCols nrToWin player1Type player2Type

Where player1Type is a string corresponding to either ‘human’ or ‘AI’ and player2Type is a string corresponding to either ‘human’ or ‘AI’.

The program displays the board each move, if human it prompts the user for input, if AI the program will automatically make its move based on the Evaluation Function.

On a 3x3 board, the program never loses. In fact, the program will never lose when nrRows = nrCols = nrToWin.
